❓ FAQ / 常见问题

Q: Hangar is asking for Screen Time / 屏幕使用时间 permission. Why?

EN: Hangar uses Apple's Screen Time API (Family Controls) to block distracting apps only during a focus session you actively start. Selection of which apps to block is entirely up to you, and is stored on-device using Apple's opaque tokens — Hangar itself cannot see which apps you picked. Blocking ends as soon as the session ends or you abandon it.

中文：Hangar 使用 Apple 的「屏幕使用时间 / Family Controls」框架，仅在你主动发起的专注会话期间屏蔽你自己选择的应用。被屏蔽的应用名单只保存在你设备本地（以 Apple 提供的加密令牌形式），Hangar 自身无法获知具体是哪些应用。会话结束或放弃后屏蔽立即解除。

Q: How do I set up Screen Time blocking? / 怎么设置屏蔽？

EN — Step by step:

  1. Open Hangar → tap the menu icon (top left) → Settings.
  2. Scroll to the Focus Shield section.
  3. If Screen Time isn't authorized yet, tap Request Authorization and grant access in the system prompt.
  4. Tap New Shield Scene, name it (e.g. "Work"), then pick which apps/categories to block in the Apple Family Activity Picker.
  5. Back on the main screen, tap Start Build → in Setup, choose a shield scene → tap Launch Build. The selected apps will be blocked until the focus session finishes or is aborted.

中文步骤：

  1. 打开 Hangar → 左上角菜单 → 设置。
  2. 找到「专注屏蔽」分区。
  3. 若尚未授权，点「请求授权」并在系统弹窗中同意。
  4. 点「新建屏蔽场景」，命名（如「工作」），在 Apple 的应用选择器中勾选要屏蔽的应用/类别。
  5. 回到主页 → 点「开始建造」→ 在任务设置里选好屏蔽场景 → 点「启动建造」。所选应用会在专注期间被屏蔽，直到完成或放弃。

Q: How do I switch the app language? / 怎么切换语言？

Settings → Language / 语言 → choose System, 简体中文, or English. Changes apply immediately, no restart needed. / 设置 → 「语言」→ 选「跟随系统」「简体中文」或「English」，立即生效。

Q: How do subscriptions work? / 订阅怎么管理？

Hangar offers a monthly subscription, a yearly subscription, and a lifetime unlock. Subscriptions auto-renew through your Apple ID. You can cancel anytime in Settings → [Your Name] → Subscriptions on iOS. / Hangar 提供月度订阅、年度订阅和终身买断。订阅通过 Apple ID 自动续费，可随时在 iOS「设置 → 你的名字 → 订阅」中取消。

Q: I can't find / tap a button. / 某个按钮按了没反应？

Please make sure you've granted Reminders / Notifications / Screen Time permission for whichever feature you're trying to use. If a button is greyed out or unresponsive, double-check the relevant permission in iOS Settings → Privacy & Security. If it still doesn't work, please email us with the device model and iOS version. / 请先确认对应权限（提醒事项 / 通知 / 屏幕使用时间）是否在 iOS「设置 → 隐私与安全性」中已经授予。如果仍然异常，请发邮件附上设备型号和系统版本。

Q: Will my data be uploaded? / 我的数据会被上传吗？

No. All your focus history, streak data, goal tree and settings are stored locally on your device. Hangar does not run any backend service. / 不会。所有专注记录、连击、蓝图与设置都只保存在你本机，Hangar 没有任何后台服务器。

Q: How do I restore my purchases on a new device? / 换设备如何恢复购买？

Sign in with the same Apple ID, then open Hangar → Settings → Restore Purchases. / 用同一个 Apple ID 登录，打开 Hangar → 设置 → 「恢复购买」。
